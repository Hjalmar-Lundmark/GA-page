Sammanfattning

Ett tillgänglighetstest har utförts för att testa tillgängligheten och användbarheten av Migrationsverket. Automatiska testverktyg har används för att hitta ställen där hemsidan kan har brutit mot tillgänglighetslagar. Web Content Accessibility Guidelines (WCAG) 2.0 är ett samlingsnamn för officiella riktlinjer inom hur webbsidor ska se ut och byggas. Detta test och verktygen utgår från dessa krav när hemsidan testas. Då Migrationsverket är en statlig hemsida så måste den hålla extra hårda krav på att alla ska kunna använda den, utifrån Svensk och Europeisk lag. Testet går igenom allt från gömda saker i bakgrunden av sidan, att skärmläsare och tangentbordsnavigering ska fungera, till hur sidan ser ut med nedsatt färgseende. Resultatet kan sammanfattas med att hemsidan är väldigt tillgänglig, med bara något problem och ställe där regler har brutits. Migrationsverkets hemsida går också att använda utan problem med nedsatt färgseende.

Abstract

This report tests the accessibility of the Swedish public web page Migrationsverket (eng: Agency of Migration), owned by the government. The test seeks to find out if the page is using correct methods and is obeying law for building web pages, while also evaluating if the page takes action combating different disabilities. The baseline of this test uses guidelines mostly from World Wide Web Consortium's Web Content Accessibility Guidelines (WCAG) 2.0. Automatic testing tools are used for determinating if the background code is correct and what impact it will have on the user. Some manual testing has also been done to evaluate some things the automatic testers could not. All and all this test shows that Migrationsverkets web page completes most legal and recommended requirements
